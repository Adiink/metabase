# Chip

Our themed wrapper around [Mantine Chip](https://v6.mantine.dev/core/chip/).

## When to use Chip

Use it in cases where you might otherwise use radio buttons, a dropdown, or a checkbox list, where a user is selected one or multiple options; but where you:

- need a more compact and visually appealing alternative to radio buttons, especially in space-constrained situations
- want to have all the options visible
- want to highlight the options visually
- have options where the text labels of each are fairly short (3â€“4 words max.)
- have fewer than ~10 total options to display
- find that a horizontal arrangement fits better in your layout

## Examples

export const DefaultTemplate = args => <Chip>I'm a chip</Chip>;

export const ChipGroupTemplate = args => (
 <Chip.Group {...args}>
  <Group>
    <Chip value="cool_ranch" >Cool Ranch</Chip>
    <Chip value="sea_sand_&_vinegar" >Sea Salt & Vinegar</Chip>
    <Chip value="garden_salsa" >Garden Salsa Sun Chips</Chip>
    <Chip value="pringles" >Pringles</Chip>
  </Group>
 </Chip.Group>
)
